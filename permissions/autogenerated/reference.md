## Default Permission

Default permissions for the plugin

- `allow-scan`
- `allow-stop-scan`
- `allow-connect`
- `allow-disconnect`
- `allow-connection-state`
- `allow-send`
- `allow-recv`
- `allow-send-string`
- `allow-recv-string`
- `allow-subscribe`
- `allow-subscribe-string`
- `allow-unsubscribe`
- `allow-scanning-state`
